## How to use this knowledge base
- Each `concepts/` file is a standalone lesson — read it cold to relearn the idea.
- Concepts are filed by category on a stable path; the week(s) a concept was touched live in
  its frontmatter and in this index, not in the folder path.
- To study, open the latest `review/<week>.md` and answer from memory before expanding each
  answer; revisit earlier weeks to space your practice.
- A rare project-specific fact worth remembering lives under **Worth remembering** in the
  relevant `sessions/` recap; everything else codebase-specific is a concept's Example.
- Run `/learning-summary` at the end of a session to keep this current.
