## How to use this knowledge base
- Each `concepts/` file is a standalone lesson — read it cold to relearn the idea.
- Concepts are filed by the week they were last touched and grouped into broader categories;
  a recurring concept moves into the current week's folder, so recent weeks show what's active.
- Codebase-specific fixes that aren't general concepts live in the relevant `sessions/` recap.
- Fill in the **My notes & links** section of a concept as you go; the skill never edits it.
- Run `/learning-summary` at the end of a session to keep this current.
